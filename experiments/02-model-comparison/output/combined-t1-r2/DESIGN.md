# Design System - CRM Dashboard

## Palette
- Page Background: #F8FAFC (slate-50)
- Surface: #FFFFFF
- Surface Subtle: #F1F5F9 (slate-100)
- Border: #E2E8F0 (slate-200)
- Border Strong: #CBD5E1 (slate-300)
- Text Primary: #0F172A (slate-900)
- Text Secondary: #475569 (slate-600)
- Text Tertiary: #94A3B8 (slate-400)
- Accent: #2563EB (blue-600)
- Accent Hover: #1D4ED8 (blue-700)
- Accent Subtle: #EFF6FF (blue-50)
- Success: #10B981 (emerald-500)
- Warning: #F59E0B (amber-500)
- Danger: #EF4444 (red-500)

## Typography
- Family: Inter, -apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif
- Scale: 11 / 12 / 13 / 14 / 16 / 18 / 20 / 24 / 30
- Weights: 400 (regular), 500 (medium), 600 (semibold), 700 (bold)
- Tabular numerals: font-variant-numeric: tabular-nums (for all metrics)

## Spacing & Shape
- Radius: 6px (inputs, buttons), 8px (cards, panels), 9999px (pills, avatars)
- Spacing unit: 4px
- Section padding: 24px (panels), 16px (cards)

## Elevation
- Card: 0 1px 2px rgba(15, 23, 42, 0.04)
- Hover: 0 1px 3px rgba(15, 23, 42, 0.06)
- NO gradients. NO glows. NO heavy shadows.

## Motion
- Default: 150ms cubic-bezier(0.4, 0, 0.2, 1)
- Slow: 200ms cubic-bezier(0.4, 0, 0.2, 1)
