React + Vite + Tailwind CSS로 SaaS 대시보드 개요 페이지를 만들어줘.

필요한 섹션:
1. 상단 통계 카드 4개 (Revenue, Users, Orders, Growth)
2. 최근 활동 목록 (시간, 사용자, 액션, 상태)
3. 월별 차트 영역 (막대 그래프 영역, 실제 차트 라이브러리 없이 CSS로만 표현)

조건:
- TypeScript 사용
- 적절한 컴포넌트 분리
- 반응형 (모바일/태블릿/데스크톱)
- Tailwind CSS만 사용
- 하나의 index.html + main.tsx에서 동작하는 구조
